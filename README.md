
## COVID19 Map challenge system

## Installation

```sh
composer install
yarn install
```

Configure your env. file with your local database credentials and local server address.

## Migrate and seed data

```sh
php artisan migrate
php artisan db:seed
```

## Running services

```sh
php artisan serve
yarn dev
```
